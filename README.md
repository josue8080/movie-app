Introduction
This project is a movie app built using HTML, CSS, and JavaScript.
It allows users to browse a collection of movies, search for specific movies, and view details about each movie.

Features
Browse a list of movies
Search for movies by title
View details of each movie (e.g., title, year, plot)

Setup Instructions
To run this project locally, follow these steps:


Navigate into the project directory: cd movie-app
Open index.html in your web browser.
Usage
Open the app in your web browser.
Browse the list of movies.
Use the search bar to find specific movies.
Click on a movie to view more details.
